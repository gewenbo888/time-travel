# Time Travel Engine · 时间旅行引擎

A bilingual atlas of time and time travel — relativity, time dilation, wormholes, the arrow of time, many-worlds branching, the three classical paradoxes (grandfather, bootstrap, predestination), consciousness and subjective time, information time, future-civilisation temporal engineering, and the question of whether time itself fundamentally flows.

Ten systems · an interactive three-lens time dial · the historical climb from Aristotle's "measure of motion" to ER=EPR · the wormhole/CTC catalogue · the entropy arrow simulator · the many-worlds branch tree · the paradox resolver · a six-lens reality analyst · a seven-axis Time Architecture radar · a recursive Future Time Engine.
